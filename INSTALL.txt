Ubuntu Linux:
You can install all required packages by doing:
  pip3 install -r requirements.txt


Windows:
  The requirements.txt file will for for everything but OpenGL.
  As of 2018 the PyOpenGL and PyOpenGL_accelerate packages in PyPI were not good.
  Instead, download the whl files for both of these for your Python version from
     https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyopengl
  and use pip to install them.


MacOS (should be 10.13 or higher):
  The requirements.txt file will for for everything but PyAudio.
  The version of PyAudio in PyPI does not compile unless you have
  certain tools installed.  Visit this page to learn how to install
  PyAudio: http://people.csail.mit.edu/hubert/pyaudio/
